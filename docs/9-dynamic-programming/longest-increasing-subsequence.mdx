#  Longest Increasing Subsequence

This page offers detailed explanations and Java solutions for problems which involve finding longest increasing subsequence, using the Dynamic Programming.


## Minimum Number of Removals to Make Mountain Array

The problem description can be found at [Leetcode 1671. Minimum Number of Removals to Make Mountain Array](https://leetcode.com/problems/minimum-number-of-removals-to-make-mountain-array/description/).



### Explanation 

To solve this problem, we identify the longest increasing subsequence from each position. For each index, we calculate the sum of the longest increasing subsequences on the left and right sides. To find the final answer, we subtract the largest sum obtained from the total length of the array.

```java
class Solution {
    public int minimumMountainRemovals(int[] nums) {
        int n = nums.length;

        // calculating Longest Increasing Subsequence from left side
        int[] dpLeft = new int[n];
        for (int i = 1; i < n; i++) {
            for (int j = 0; j < i; j++) {
                if (nums[j] < nums[i]) {
                    dpLeft[i] = Math.max(dpLeft[i], dpLeft[j] + 1);
                }
            }
        }

        // calculating Longest Increasing Subsequence from right side
        int[] dpRight = new int[n];
        for (int i = n - 2; i >= 0; i--) {
            for (int j = n - 1; j > i; j--) {
                if (nums[j] < nums[i]) {
                    dpRight[i] = Math.max(dpRight[i], dpRight[j] + 1);
                }
            }
        }

        // calculate the maximum sum of the Longest Increasing Subsequence 
       // from the left and right sides.
        int max = 0;
        for (int i = 1; i < n - 1; i++) {
            if (dpLeft[i] > 0 && dpRight[i] > 0) {
                max = Math.max(max, dpLeft[i] + dpRight[i] + 1);
            }
        }

        // substract max value from total length of array
        return n - max;
    }
}
```
